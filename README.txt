COMMUNITY MANAGERS GROUP.

La página web presentada, es una web de community managers que se encargan de potenciar marcas, empresas,
emprendimientos, etc. 

Podrán iniciar sesión únicamente los miembros del staff. Dentro del staff hay grupos definidos, los cuales cada uno
trabajan sobre cierto proyecto. Estos grupos tienen un líder que propone el plan de trabajo para abordar al cliente. 

Si no se inicia sesión lo único que se visualiza en la web es la info de Community Managers Group: propósito,
objetivos, propuesta de trabajo, etc. 

Si inicia sesión un líder, este podrá editar grupos, eliminar grupos, y armar nuevos grupos. 
Si inicia sesión un miembro del grupo, este podrá únicamente editar su propio usuario y datos.
